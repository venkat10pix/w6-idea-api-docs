Introduction

Welcome to the Idea Tracker API. This API allows developers to programmatically manage creative concepts, track their progress, and collaborate on brainstorming sessions. Our RESTful API is designed to be intuitive, secure, and fast.

Base URL: https://api.ideatracker.io/v1

01 Authentication

All API requests require an API key to be included in the header.

Request Header Required 
X-API-Key: your_api_key_here
GET

/api/ideas

Retrieves a paginated list of all ideas currently stored in your account database. Use query parameters to filter by category or status.

Response Example

200 OK — JSON 
{
  "status": "success",
  "data": [
    {
      "id": "idea_9281",
      "title": "AI Personal Stylist",
      "description": "A mobile app that uses computer vision to recommend outfits.",

POST

/api/ideas

Create a new idea entry in your workspace. Requires a JSON body with title and description fields.

Request Body

{
  "title": "Smart Compost Bin",
  "description": "IoT compost bin with heat tracking.",
  "category": "Hardware"
}

Response Example

{
  "id": "idea_1105",
  "message": "Idea created successfully"
}

# Idea Tracker API

A professional tool for capturing, sharing, and tracking team brilliance. This project includes a premium frontend for idea management and a fully documented API.

## 🚀 Features
- **Instant Idea Submission**: Quick form to capture sparks of innovation.
- **Idea Board**: A dynamic gallery to visualize team contributions.
- **API Documentation**: Comprehensive documentation found in the `/docs` folder.

## 📖 API Documentation
The API documentation is built as a static site and is located at `/docs/index.html`. It can be deployed via GitHub Pages.

## 🛠️ Tech Stack
- **Frontend**: HTML5, Vanilla JavaScript, Custom CSS.
- **Documentation**: Tailwind CSS via CDN.

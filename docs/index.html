Introduction

Welcome to the Idea Tracker API documentation. This API allows developers to programmatically interact with the Idea Tracker platform, enabling the creation, retrieval, and management of brilliant team ideas.

Base URL

https://api.ideatracker.io/v1

Authentication

All API requests must be authenticated using a Bearer token in the Authorization header.

Example Request Header 
Authorization: Bearer YOUR_API_TOKEN

Keep your API tokens secure. Do not share them or commit them to public version control systems.

API Endpoints

GET

List all Ideas

Retrieves a list of all ideas submitted to the board, sorted by most recent first.

Parameters

None required.

Example Response

200 OK 
[
  {
    "id": "7f12a",
    "title": "Implement AI Chatbot",
    "description": "Add a smart assistant to help users...",
    "createdAt": "2024-03-20T14:30:00Z"
  },
  {
    "id": "9b34c",
    "title": "DarkMode Toggle",
    "description": "Allow users to switch between themes.",
    "createdAt": "2024-03-19T09:15:00Z"
  }
]
POST

Create a New Idea

Submits a new idea to the gallery. Validates title (max 50 chars) and description (max 500 chars).

Request Body

{
  "title": "Global Search",
  "description": "Add a search bar to filter ideas."
}

Response

201 Created 
{
  "success": true,
  "id": "8e45f",
  "message": "Idea successfully added."
}
GET

Get Idea Details

Retrieves complete information for a specific idea by its unique ID.

Path Variables

Property Description
id Unique identifier of the idea.

Example Response

{
  "id": "7f12a",
  "title": "Implement AI Chatbot",
  "description": "Add a smart assistant...",
  "createdAt": "2024-03-20T14:30:00Z"
}
DELETE

Remove an Idea

Deletes a specific idea from the tracking board. This action is irreversible.

204 No Content
Empty response body.

Request Header REQUIRED 
Authorization: Bearer your_api_key_here

REST Endpoints

GET

/api/ideas

Retrieve a paginated list of all created ideas.

This endpoint returns a list of idea objects. You can filter the results by category or sort them by date using query parameters.

Query Parameters

category String - Filter by specific category
limit Integer - Number of results (default: 20)

POST

/api/ideas

Create a new genius idea in the system.

Send a JSON payload with the details of your idea. The title and category fields are mandatory.

Rate Limit

Default rate limit is 100 requests per minute. Contact sales for higher quotas.
